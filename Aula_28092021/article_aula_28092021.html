Google Chrome

Mozila Firefox

Mozila Firefox é um navegador web opensource desenvolvido pela Mozila. O Firefox tem sido o segundo navegador web mais popular desde janeiro, 2018.

Microsoft Edge é um navegador web lançado pela Microsoft, em 2015. O Microsoft Edge substituiu o Internet Explorer.
